Hello! Here's how to use my MIDI player


First, obviously you need Python
This version was tested in Python 3.13.9, but other versions should work too

Next, you need to run setup_win.bat
It will install all the necessary packages for this program to run.

Now, open up settings.py with any text editor
You can change some settings there.

And finally, you can run the program by running run.bat
